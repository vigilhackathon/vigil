# VIGIL — the agent that watches the waiting room

An agentic monitor for the gap between ED triage and being seen. Structured voice/text check-ins with waiting-room patients (ESI 3–5), delta-anchored to each patient's own baseline, with a deterministic safety guardrail the model cannot override — tiering patients **Routine / Watch / Escalate**, closing the loop with nurse acknowledgment, and generating an interval handoff report when the patient is called back.

Built for **The Future of Agentic AI in Healthcare** hackathon (Abridge × Anthropic × Lightspeed), San Francisco, July 18, 2026.
